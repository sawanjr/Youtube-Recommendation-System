YouTube Recommendation System

Information

User ID: This is a unique identifier for each user. (You can assume it as a User's Account in YouTube.)

Item ID: This corresponds to the video or item you want a rating for. (It is a video that the user will interact with.)

Predicted Rating: This is a score from 1 to 5 that suggests how much you will like the video.

Based on predicted ratings, the system can suggest videos that match your preferences.

If the predicted rating is high (e.g., 4.0 or above), it means the model believes the user will likely enjoy the item, and thus, it should be recommended to them.

If the predicted rating is low, it suggests the user probably won’t like the item, so it wouldn’t be recommended.
